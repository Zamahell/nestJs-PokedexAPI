#Ejecutar en desarollo

1. Clonar el repositorios
2. Ejecutrar el comando
```
yarn install
```
3. Tener Nest Cli instalado
```
npm i -g @nestjs/cli
```

4. Levantar la base de datos
```
docker-compose up -d
```
5. Reconstruir la base de datos con el SEED

```
http://localhost:3000/api/v2/seed
```

## Stack Usado
* MongoDB
* Nest
